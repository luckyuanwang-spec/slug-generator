# Slug Generator

Free online URL slug generator to convert any text into SEO-friendly slugs.

**Live Demo**: https://slug-generator-phi.vercel.app/

## Features

- Convert any text to URL-friendly slugs
- One-click presets for blog posts, products, categories
- Toggle options: lowercase, remove stop words, trim dashes
- One-click copy to clipboard
- Bilingual support (English/Chinese)
- SEO optimized with meta tags
- Mobile responsive

## Tech Stack

- Pure HTML/CSS/JavaScript
- No dependencies required
- Vercel hosting (free)
